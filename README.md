# Netflix-PowerBI-Dashboard
Interactive Netflix Dashboard built using Power BI. Includes analysis of content type, ratings, release years, countries, and genres.

Project Overview:<br>
This project presents an interactive Netflix Dashboard built using Microsoft Power BI.<br>
The dashboard provides insights into Netflix's content library, helping users analyze trends in movies and TV shows based on ratings, genres, countries, release years, and content types.<br>

Objectives:<br>
-Analyze Netflix content distribution.<br>
-Compare Movies and TV Shows.<br>
-Identify trends in content releases over time.<br>
-Explore content ratings and genres.<br>
-Visualize country-wise content availability.<br>

Tools & Technologies:<br>
-Power BI Desktop<br>
-DAX (Data Analysis Expressions)<br>
-Power Query<br>
-Data Visualization<br>

Dashboard Features:<br>
Total Movies and TV Shows<br>
Content Distribution by Type<br>
Titles Added by Year<br>
Ratings Analysis<br>
Genre Analysis<br>
Country-wise Content Distribution<br>
Interactive Filters and Slicers<br>

Key Insights:<br>
Movies constitute the majority of Netflix content.<br>
Content additions increased significantly in recent years.<br>
Certain ratings dominate the platform's catalog.<br>
The United States contributes the largest share of content.<br>
Drama and International genres are among the most popular categories.<br>
